# rollup-plugin-type-as-json-schema

> A Rollup Plugin for Import TS Types as JSON Schema.

## Usage

```bash
npm i -D rollup-plugin-type-as-json-schema
```

or

```bash
pnpm add -D rollup-plugin-type-as-json-schema
```

## Workflow

```bash
# develop
npm start

# build
npm run build

# test
npm test
```
